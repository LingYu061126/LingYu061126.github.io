3.1 变压器的空载运行

一、基本概念

空载运行定义：一次绕组接额定频率、额定电压的交流电源，二次绕组开路时的运行状态。

空载电流：变压器空载运行时，一次绕组流过的电流。

二、空载运行时的物理过程

空载运行过程：

空载电流 → 空载磁动势 → 空载磁通 → 主磁通和漏磁通 → 主磁路和漏磁路

主磁通与漏磁通的区别：

特性 主磁通 漏磁通
磁路性质 铁磁材料，有磁饱和，与空载电流非线性 非铁磁材料，无磁饱和，与空载电流线性
占比 占总磁通绝大部分 很小，仅占 0.1%~0.2%
功能 传递能量 只感应电动势，不传递功率

  • 瞬时值：e1=−N1dϕdt=ωN1ϕmsin⁡(ωt−π2)=E1msin⁡(ωt−π2)e_1 = -N_1\frac{d\phi}{dt} = \omega N_1\phi_m\sin(\omega t - \frac{\pi}{2}) = E_{1m}\sin(\omega t - \frac{\pi}{2})e1​=−N1​dtdϕ​=ωN1​ϕm​sin(ωt−2π​)=E1m​sin(ωt−2π​)
  • 有效值：E1=4.44fN1ϕmE_1 = 4.44fN_1\phi_mE1​=4.44fN1​ϕm​

二次绕组感应电动势：

  • 有效值：E2=4.44fN2ϕmE_2 = 4.44fN_2\phi_mE2​=4.44fN2​ϕm​

重要结论：

  1. 感应电动势滞后产生该电动势的磁通 90°
  2. 感应电动势有效值与频率、绕组匝数、主磁通最大值成正比

其中 x1σx_{1\sigma}x1σ​ 为一次侧漏电抗（常数）

四、电动势方程与变比

空载运行时电动势方程：

U˙1=−E˙1+jI˙0x1σ+I˙0r1=−E˙1+I˙0Z1\dot{U}_1 = -\dot{E}_1 + j\dot{I}_0x_{1\sigma} + \dot{I}_0r_1 = -\dot{E}_1 + \dot{I}_0Z_1 U˙1​=−E˙1​+jI˙0​x1σ​+I˙0​r1​=−E˙1​+I˙0​Z1​

简化（忽略一次绕组漏阻抗压降）：

U˙1≈−E˙1\dot{U}_1 \approx -\dot{E}_1 U˙1​≈−E˙1​

U1≈E1=4.44fN1ϕmU_1 \approx E_1 = 4.44fN_1\phi_m U1​≈E1​=4.44fN1​ϕm​

关键结论：

  • 外施电压与感应电动势大小相等，相位相反（反电动势）
  • 电源频率和匝数为常数时，主磁通最大值与电源电压成正比
  • 电源电压一定，主磁通最大值一定，产生主磁通的励磁磁动势也一定

五、空载电流

空载电流的组成：

  • 无功分量（磁化电流） >> 有功分量（铁耗电流）
  • 空载电流为无功性质，又称激磁电流或励磁电流

空载电流大小：

  • 约为额定电流的2%以内
  • 变压器容量越大，空载电流百分数越小

国家标准（GB/T 6451-2015）：

  • 6kV、10kV配电，200kVA以下：< 1.5%
  • 200kVA以上，2500kVA为0.4%
  • 500kV电压等级334MVA以上：为0.1%

空载电流波形：

条件 磁通波形 空载电流波形
磁路不饱和 正弦波 正弦波
磁路饱和 正弦波 尖顶波（含3、5、7次等奇次谐波）

六、空载损耗

组成：

  • 一次绕组铜耗 pCup_{Cu}pCu​（可忽略）
  • 铁心铁耗 pFep_{Fe}pFe​（磁滞损耗和涡流损耗）

近似计算：

p0≈pFep_0 \approx p_{Fe} p0​≈pFe​

大小：一般占额定容量的0.2%~1%

七、等效电路与相量图

励磁阻抗：

Zm=rm+jxmZ_m = r_m + jx_m Zm​=rm​+jxm​

  • xmx_mxm​ — 励磁电抗（对应主磁通的电抗）
  • rmr_mrm​ — 励磁电阻（对应铁耗的等效电阻）

铁耗计算：

pFe=I02rmp_{Fe} = I_0^2r_m pFe​=I02​rm​

相量图特点：

  • 空载时，功率因数很低，感性，约0.1~0.2左右
  • 从电网吸收功率极少，主要为一次绕组电阻损耗（铁耗）和很小的铁耗分量

八、物理量正方向规定

  1. 电源电压正方向：A→X，一次侧电流正方向与电源正方向一致（A→X）
  2. 磁动势正方向：与产生该磁动势的电流正方向之间符合右手螺旋关系
  3. 磁通正方向：与磁动势的正方向一致
  4. 感应电动势正方向（电位升高方向）：与产生该电动势的磁通正方向之间符合右手螺旋关系
  5. 二次侧：按"电源惯例"，绕组电动势看作电源电动势，a-x之间接负载时，二次侧电流正方向与感应电动势正方向一致，而负载端电压正方向与电流正方向一致

九、案例分析要点

气隙增大时的影响（直流 vs 交流）：

类型 磁通 电感 电流
直流 不变 ↓ 减小 不变（电流由电源电压和电阻决定）
交流 不变（U1=4.44fN1ϕmU_1 = 4.44fN_1\phi_mU1​=4.44fN1​ϕm​，电压一定则磁通一定） ↓ 减小 ↑ 增大（磁动势要增大以维持磁通）
