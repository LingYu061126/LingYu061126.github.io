电机学（上）知识点汇总

2026年春季考试范围整理，参考《电机学》教材及《电机学学习指导》

第1章 磁路及磁路定律

1.1 磁场的基本物理量

物理量 符号 定义 SI单位
磁感应强度（磁通密度） B 磁场内某点磁场强弱的物理量 特斯拉(T)，1T=10⁴Gs
磁通 Φ 垂直通过截面的磁力线总量 韦伯(Wb)，1Mx=1Wb
磁导率 μ 表示物质导磁性能的参数 亨/米(H/m)
磁场强度 H B/μ，与介质无关 安/米(A/m)

基本关系：

  • Φ = BA（均匀磁场），B = Φ/A，故磁感应强度又称磁通密度
  • H = B/μ，磁场强度只与电流及分布有关，与磁介质无关，引入仅为简化计算
  • 真空磁导率 μ₀ = 4π×10⁻⁷ H/m
  • 相对磁导率 μᵣ = μ/μ₀（无量纲，非铁磁物质μᵣ≈1，铁磁物质μᵣ >> 1，为10~10⁵倍）

磁路与电路的类比区别：

  • 电路中导体与绝缘体电导率之比达10¹⁶，电流沿导体流通
  • 磁路中铁磁与非铁磁物质磁导率之比仅10³~10⁵，没有绝对的"磁绝缘体"
  • 电路中导体电阻可认为是常数（线性），而铁磁材料磁导率随饱和程度变化（非线性）

1.2 基本电磁定律

（1）安培环路定律（全电流定律）

沿任一闭合路径磁场强度H的线积分等于与该路径交链的电流代数和：

∮H⃗⋅dl⃗=∑I\oint \vec{H} \cdot d\vec{l} = \sum I ∮H⋅dl=∑I

简化形式：ΣHl = ΣNi（N为线圈匝数）

（2）法拉第电磁感应定律

e=−dΨdt=−NdΦdte = -\frac{d\Psi}{dt} = -N\frac{d\Phi}{dt} e=−dtdΨ​=−NdtdΦ​

其中 Ψ = NΦ 为磁链。

  • 变压器电动势：线圈与磁场相对静止，磁通随时间变化
  • 运动电动势：磁场恒定，线圈运动切割磁力线，e = Blv
  • 自感电动势：e_L = -L(di/dt)，L为自感系数
  • 互感电动势：e_M1 = -M(di₂/dt)，M为互感系数

（3）毕-萨电磁力定律

f=Blif = Bli f=Bli

受力方向由左手定则判定。

（4）磁路欧姆定律

F=ΦRmF = \Phi R_m F=ΦRm​

  • F：磁动势（单位A），F = Ni
  • R_m：磁阻，R_m = l/(μA)
  • Λ_m：磁导，Λ_m = 1/R_m

（5）磁路基尔霍夫第一定律

∮SB⃗⋅ds⃗=0\oint_S \vec{B} \cdot d\vec{s} = 0 ∮S​B⋅ds=0

穿出任一闭合面的总磁通等于进入的总磁通。

（6）磁路基尔霍夫第二定律

∑F=∑Hl=∑NI\sum F = \sum Hl = \sum NI ∑F=∑Hl=∑NI

沿闭合磁路的总磁动势等于各段磁压降的代数和。

1.3 铁磁材料

分类：

  • 软磁材料：磁滞回线窄，矫顽力小，剩磁小。如硅钢片、铸钢、铸铁。用于电机和变压器铁心。
  • 硬磁材料（永磁材料）：磁滞回线宽，矫顽力大，剩磁大。如铁氧体、钕铁硼。用于永磁电机。

硅钢片特点：磁导率高、损耗小、有饱和现象。电机和变压器铁心常采用硅钢片叠成。

硅钢片分类：

  • 按含硅量：低硅片（≤2.8%，电机用）和高硅片（2.8%~4.8%，变压器用）
  • 按加工工艺：热轧硅钢片和冷轧硅钢片（冷轧性能更优，我国已停止使用热轧）
  • 冷轧又分：无取向硅钢片（电机用，磁各向同性）和取向硅钢片（变压器用，轧制方向磁性能优越）
  • 取向硅钢铁损仅为热轧的1/2，磁导率为热轧的2.5倍
  • 非晶合金：磁导率比取向硅钢更高，损耗更低，用于低损耗变压器，空载损耗可降50%~80%

铁心损耗（铁损）：

  • 磁滞损耗：交变磁场中磁畴反复摩擦产生。与频率f、磁通密度幅值B_m、材料体积有关。
    • 经验公式：P_h = C_h × f × B_m^n × V（n取1.6~2.3，估算取2）
  • 涡流损耗：变化磁通在铁心中感应涡流产生。与频率、电阻率、B_m、硅钢片厚度有关。
    • 经验公式：P_e = C_e × f² × B_m² × d² × V（d为硅钢片厚度）
  • 总铁损：P_Fe = P_h + P_e ≈ C_Fe × f^β × B_m² × V（β=1.2~1.6）
  • 铁心损耗与磁通密度、频率、材料体积、硅钢片厚度等有关；交变磁通才有铁耗，恒定磁通无铁耗
  • 减小铁损措施：采用薄硅钢片叠成、片间涂绝缘、加入硅提高电阻率

磁化曲线：

  • 起始磁化曲线：从零磁化状态开始，分四段（起始段→快速增长段→饱和段→深度饱和段）
  • 膝点：磁化曲线的拐弯处，电机设计时额定磁通密度通常选在膝点附近
  • 基本磁化曲线：各磁滞回线顶点连线，与起始磁化曲线差别不大
  • 工程计算采用基本磁化曲线（直流磁路计算）
  • 磁饱和：随H增加而B增加很小的现象，铁磁材料的重要特性
  • 磁滞现象：B的变化落后于H的变化；剩磁B_r和矫顽力H_c是两个重要参数

1.4 磁路计算要点

  • 铁心磁路是非线性的（存在饱和），不能使用叠加原理
  • 直流磁路：气隙增大→R_m增大→Φ减小→电感减小→电流增大
  • 交流磁路：电压不变时，气隙增大→R_m增大→但Φ基本不变（由电压决定）→电流增大
  • 磁路中叠加原理仅在线性情况下可用

第2章 电力变压器的基本结构

2.1 变压器的用途和分类

用途：将一种电压等级的交流电能转换为同频率另一种电压等级的交流电能。

分类：

  • 按用途：电力变压器、特种变压器（整流、矿用、船用等）
  • 按相数：单相变压器、三相变压器
  • 按绕组数：双绕组、三绕组、自耦变压器
  • 按冷却方式：油浸式（自然循环、强迫循环）、干式

2.2 变压器的基本结构及部件

核心组成（油浸式变压器）：

器身（铁心+绕组）：

  • 铁心：磁路的通道，由硅钢片叠压而成，是变压器最基本的部件
    • 铁心柱：套装绕组的部分
    • 铁轭：连接各铁心柱形成闭合磁路
    • 心式结构：绕组包围铁心柱（国产变压器主要采用）
    • 壳式结构：铁心包围绕组（国外大型变压器常用）
  • 绕组（线圈）：电路部分，是变压器的核心部件
    • 一次绕组（原边/初级）：接电源
    • 二次绕组（副边/次级）：接负载
    • 同心式绕组：高低压绕组同心套装（最常用）
    • 交叠式绕组：高低压绕组交替放置

油箱及附件：

  • 变压器油：起冷却和绝缘介质双重作用
  • 油箱：外壳，起冷却、散热和保护作用
  • 储油柜（油枕）：调节油量，减小油与空气接触面积
  • 安全气道（防爆管）：内部故障时释放压力
  • 气体继电器（瓦斯继电器）：检测内部故障
  • 分接开关：调节电压比（无励磁调压/有载调压）
  • 绝缘套管：引出绕组端线，起绝缘作用
  • 吸湿器：吸收进入储油柜空气中的水分
  • 温度计：监测油温

2.3 变压器的损耗、温升及冷却

损耗分类：

  • 铜损（负载损耗）：绕组电阻产生的I²R损耗
  • 铁损（空载损耗）：磁滞损耗+涡流损耗

冷却方式：

  • 油浸自冷（ONAN）
  • 油浸风冷（ONAF）
  • 强迫油循环（OFAF/ODAF）
  • 干式自冷（AN/AF）

2.4 电力变压器的型号及额定值

主要额定值：

  • 额定容量 S_N（kVA或MVA）
  • 额定电压 U_N（一次侧/二次侧，kV）
  • 额定电流 I_N = S_N / (√3 × U_N)（三相）
  • 额定频率 f_N（50Hz）
  • 联结组标号（如Yyn0、Dyn11等）

电压比（变比）： k = N₁/N₂ ≈ U₁/U₂

第3章 变压器基本运行原理

3.1 单相变压器的空载运行

空载运行：一次绕组接交流电源，二次绕组开路。

空载时的电磁关系：

  • 一次侧电流 I₀（空载电流）很小，约为额定电流的2%~10%
  • 空载电流包含：
    • 磁化电流 I_m：建立主磁通，无功分量（占主要部分）
    • 铁损电流 I_Fe：提供铁心损耗，有功分量（很小）
  • 空载电流相量：İ₀ = İ_Fe + İ_m，超前主磁通一个小角度α（接近90°），说明空载功率因数很低
  • 主磁通Φ（占总磁通99%以上）同时交链一、二次绕组，传递能量
  • 漏磁通Φ_σ（仅占0.1%~0.2%）只交链一次绕组，不传递功率
  • 主磁通性质：经铁心闭合，可能出现饱和，与电流非线性
  • 漏磁通性质：经空气闭合，不饱和，与电流成线性关系
  • 额定电压下铁心一般不饱和（冷轧硅钢片B不超过1.75T），空载电流近正弦波
  • 空载损耗 P₀ ≈ 铁损 P_Fe（空载铜损可忽略），一般占额定容量0.2%~1%

感应电动势：

E1=4.44fN1ΦmE_1 = 4.44 f N_1 \Phi_m E1​=4.44fN1​Φm​

E2=4.44fN2ΦmE_2 = 4.44 f N_2 \Phi_m E2​=4.44fN2​Φm​

其中 Φ_m 为主磁通幅值，4.44 = 2π/√2。推导：设Φ = Φ_m sinωt，则 e = -N dΦ/dt = -ωNΦ_m cosωt = E_m sin(ωt-90°)，有效值 E = E_m/√2 = ωNΦ_m/√2 = 2πfNΦ_m/√2 = 4.44fNΦ_m。

感应电动势相位：滞后产生它的主磁通90°

电压平衡方程：

U˙1=−E˙1+I˙0Z1\dot{U}_1 = -\dot{E}_1 + \dot{I}_0 Z_1 U˙1​=−E˙1​+I˙0​Z1​

其中 Z₁ = R₁ + jX_σ₁ 为一次绕组漏阻抗。由于空载时 I₀ 很小，I₀Z₁ 可忽略，近似有 U₁ ≈ E₁ = 4.44fN₁Φ_m

重要结论：电源电压U₁一定时，主磁通Φ_m也一定，与是否带负载无关。

漏电抗：漏磁通路径为非铁磁材料（线性），漏电动势可用漏电抗压降表示：Ė_σ1 = -jİ₀X_σ₁

励磁阻抗：主磁通在铁心中引起铁耗，需用阻抗表示：-Ė₁ = İ₀Z_m = İ₀(R_m + jX_m)

  • R_m：励磁电阻，对应铁心损耗的等效电阻（P_Fe = I₀²R_m）
  • X_m：励磁电抗，对应主磁通的电抗
  • 励磁阻抗不是常数，随铁心饱和程度增加而减小
  • 空载时：R_m >> R₁，X_m >> X_σ₁

空载运行等效电路：一次绕组漏阻抗Z₁与励磁阻抗Z_m串联

变比：

k=E1E2=N1N2≈U1U2k = \frac{E_1}{E_2} = \frac{N_1}{N_2} \approx \frac{U_1}{U_2} k=E2​E1​​=N2​N1​​≈U2​U1​​

三相变压器变比为相电动势之比（需将线电压换算为相电压）：

  • Yy连接：k = U₁/√3 ÷ U₂/√3 = U₁/U₂
  • Yd连接：k = U₁/√3 ÷ U₂ = U₁/(√3U₂)
  • Dy连接：k = U₁ ÷ U₂/√3 = √3U₁/U₂

3.2 单相变压器的负载运行

负载运行：一次绕组接电源，二次绕组接负载阻抗。

磁动势平衡：

F˙1=F˙0+F˙2即N1I˙1=N1I˙0+(−N2I˙2)\dot{F}_1 = \dot{F}_0 + \dot{F}_2 \quad 即 \quad N_1\dot{I}_1 = N_1\dot{I}_0 + (-N_2\dot{I}_2) F˙1​=F˙0​+F˙2​即N1​I˙1​=N1​I˙0​+(−N2​I˙2​)

即 İ₁ = İ₀ + İ₁L，其中 İ₁L = -İ₂/k 为一次侧电流的负载分量。

物理意义：一次侧磁动势由两部分组成：①励磁分量İ₀（建立主磁通，近似不变）；②负载分量İ₁L（平衡二次侧磁动势，随负载变化）。不管负载如何变化，主磁通基本恒定（由电源电压决定）。

电流关系： İ₁/İ₂ ≈ N₂/N₁ = 1/k（忽略İ₀时）

绕组折算（归算）：将二次侧参数折算到一次侧，使变比k=1

  • E₂’ = kE₂，U₂’ = kU₂
  • I₂’ = I₂/k
  • R₂’ = k²R₂，X_σ₂’ = k²X_σ₂，Z₂’ = k²Z₂

漏磁通与漏抗：

  • 一次漏磁通 Φ_σ1 → 漏电抗 X_σ1（线性，常数）
  • 二次漏磁通 Φ_σ2 → 漏电抗 X_σ2（线性，常数）

负载时的电压平衡方程（基本方程式）：

U˙1=−E˙1+I˙1(R1+jXσ1)\dot{U}_1 = -\dot{E}_1 + \dot{I}_1(R_1 + jX_{\sigma1}) U˙1​=−E˙1​+I˙1​(R1​+jXσ1​)

E˙2=U˙2+I˙2(R2+jXσ2)\dot{E}_2 = \dot{U}_2 + \dot{I}_2(R_2 + jX_{\sigma2}) E˙2​=U˙2​+I˙2​(R2​+jXσ2​)

−E˙1=I˙0Zm-\dot{E}_1 = \dot{I}_0 Z_m −E˙1​=I˙0​Zm​

E˙1=kE˙2\dot{E}_1 = k\dot{E}_2 E˙1​=kE˙2​

I˙1=I˙0+I˙1L\dot{I}_1 = \dot{I}_0 + \dot{I}_1L I˙1​=I˙0​+I˙1​L

T型等效电路（折算后）：

  • 一次侧：R₁ + jX_σ₁
  • 励磁支路：R_m + jX_m（并联）
  • 二次侧（折算后）：R₂’ + jX_σ₂’ + Z_L’

简化等效电路：将励磁支路移到端部（因İ₀很小），近似为 R_k + jX_k 与负载串联

重要结论：

  • 忽略漏阻抗压降时，U₁ ≈ E₁ = 4.44fN₁Φ_m，即主磁通由电源电压决定
  • 主磁通大小只由电源电压决定，与是否带负载及负载大小无关（这一点对分析变压器运行十分重要）
  • 励磁阻抗Z_m = R_m + jX_m不是常数，随铁心饱和程度变化（饱和越深，Z_m越小）

3.3 标幺值

标幺值 = 实际值 / 基值

选取基值的原则：

  • 通常取额定值作为基值
  • U* = U/U_N，I* = I/I_N
  • S* = S/S_N，Z* = Z/Z_N

标幺值的优点：

  • 不同容量变压器参数便于比较
  • 简化计算（如Y-△变换）
  • 额定时各量标幺值为1

3.4 变压器等效电路参数测定

通过空载试验和短路试验确定等效电路参数。

空载试验（测铁损和励磁参数）

  • 接法：低压侧加压，高压侧开路
  • 测量：空载电压 U₀、空载电流 I₀、空载功率 P₀
  • 计算：
    • 励磁电阻 R_m = P₀/I₀²
    • 励磁阻抗 Z_m = U₀/I₀
    • 励磁电抗 X_m = √(Z_m² - R_m²)
    • P₀ ≈ 铁损 P_Fe

短路试验（测铜损和短路参数）

  • 接法：高压侧加压，低压侧短路
  • 测量：短路电压 U_k、短路电流 I_k、短路功率 P_k
  • 计算：
    • 短路电阻 R_k = P_k/I_k²
    • 短路阻抗 Z_k = U_k/I_k
    • 短路电抗 X_k = √(Z_k² - R_k²)
    • P_k ≈ 铜损 P_Cu

短路电压（阻抗电压）u_k%：额定电流下短路阻抗上的压降百分比，是变压器重要参数。

3.5 变压器的运行特性

电压调整率：

ΔU%=U20−U2U20×100%\Delta U\% = \frac{U_{20} - U_2}{U_{20}} \times 100\% ΔU%=U20​U20​−U2​​×100%

近似公式：

ΔU%=β(Rk∗cos⁡φ2+Xk∗sin⁡φ2)×100%\Delta U\% = \beta(R_k^* \cos\varphi_2 + X_k^* \sin\varphi_2) \times 100\% ΔU%=β(Rk∗​cosφ2​+Xk∗​sinφ2​)×100%

第4章 三相变压器的磁路和电路系统

4.1 三相变压器的磁路系统

两种基本型式：

  1. 三相组式（三相变压器组）

    • 由三台单相变压器按一定连接方式组合而成
    • 三相磁路彼此独立，各不相关，三相磁路完全相同
    • 优点：运输方便、备用容量小、各相可独立检修
    • 缺点：消耗材料多、占地面积大、成本高
    • 三相磁通对称时，三相空载电流也对称

  2. 三相芯式（三相三柱式）

    • 由三个单相铁心合并演变而来（中间心柱因三相磁通之和为零而省去）
    • 三相磁路互相关联，中间相磁路较短（磁阻较小）
    • 优点：省材料、体积小、效率高、成本低
    • 缺点：运输较困难
    • 实际中广泛采用
    • 注意：三相空载电流不完全对称（中间相磁路短，磁阻小，空载电流较小）

4.2 三相变压器的电路系统——绕组连接方式和联结组标号

绕组连接方式：

连接方式 特点
Y（星形） 有中性点，线电压=√3×相电压，线电流=相电流
D（三角形） 无中性点，线电压=相电压，线电流=√3×相电流

三相变压器变比计算（重要！）：

  • Yy连接：k = U₁N/(√3 × U₂N)
  • Yd连接：k = √3 × U₁N/U₂N
  • Dy连接：k = U₁N/(√3 × U₂N)
  • 注意：标注的额定电压为线电压，求变比时必须换算成相电压之比

常见联结组：

  • Yyn0：高压Y，低压yn，中性线引出，相位差0°
  • Dyn11：高压D，低压yn，中性线引出，相位差330°（11点钟）
  • YNd11：高压YN，低压d

联结组标号确定方法：

  1. 画出高压侧相电压相量图
  2. 画出低压侧相电压相量图
  3. 将高压侧A点与时钟12点重合
  4. 低压侧a点所指的小时数即为联结组号

4.3 绕组连接方式及磁路系统对电动势波形的影响

  • Yy连接：三次谐波磁通不能在铁心中流通，感应电动势接近正弦波
  • Yd连接：三次谐波电流在三角形中流通，改善电动势波形
  • 三相芯式磁路：三次谐波磁通经油箱壁等路径流通，附加损耗大
  • D连接有利于改善波形

第5章 三相变压器的运行

5.1 三相变压器的并联运行

并联运行条件：

  1. 变比相等（电压比相同）
  2. 联结组标号相同
  3. 短路阻抗标幺值相等（阻抗电压相同）
  4. 短路阻抗的阻抗角相同

不满足条件的后果：

  • 变比不等：产生环流，增加损耗
  • 联结组不同：产生很大环流，可能烧毁变压器
  • 短路阻抗不等：负载分配不均匀，容量不能充分利用

负载分配：

SISIN:SIISIIN=1ZkI∗:1ZkII∗\frac{S_I}{S_{I_N}} : \frac{S_{II}}{S_{II_N}} = \frac{1}{Z_{kI}^*} : \frac{1}{Z_{kII}^*} SIN​​SI​​:SIIN​​SII​​=ZkI∗​1​:ZkII∗​1​

阻抗标幺值小的变压器先达到满载。

5.2 三相变压器的不对称运行

分析方法：对称分量法

将不对称的三相量分解为：

  • 正序分量：与正常相序相同
  • 负序分量：与正常相序相反
  • 零序分量：三相同相位

Yyn连接变压器带单相负载：

  • 零序电流可以在低压侧流通
  • 高压侧无零序电流（Y无中线）
  • 中性点偏移，导致三相电压不对称
  • 输出电压波形可能畸变

Dyn11连接的优势：

  • D连接为零序电流提供通路
  • 带单相负载能力强
  • 三相电压对称性好

第8章 交流绕组的构成及其电动势和磁动势

8.1 交流电机的基本工作原理

交流电机（同步电机和异步电机）的共同部分——交流绕组。旋转磁场是交流电机工作的基础。

8.2 交流绕组的构成原则和分类

构成原则：

  1. 产生接近正弦分布的磁动势
  2. 三相对称绕组产生旋转磁场
  3. 导体电流密度均匀，散热良好
  4. 用铜量少，绝缘可靠

分类：

  • 按层数：单层绕组、双层绕组
  • 按绕法：叠绕组、波绕组

8.3 交流绕组的基本术语

术语 符号 含义 公式
磁极对数 p 主磁极的对数 -
电角度 - 一对磁极占360°电角度 电角度 = p × 机械角度
极距 τ τ 相邻异性磁极间的圆周距离 τ = πD/(2p) 或 τ = Z/(2p)（用槽数表示）
节距 y y 线圈两有效边之间的距离 整距y=τ，短距y<τ，长距y>τ
槽距角 α α 相邻两槽之间的电角度 α = p×360°/Z
每极每相槽数 q q 每相在每极下平均占有的槽数 q = Z/(2pm)，m为相数
相带 - 每相在每极下所占的电角度范围 60°相带（常用）或120°相带
绕组系数 k_w k_w 绕组的等效系数 k_w = k_d × k_p
分布系数 k_d k_d 反映绕组分布的影响 k_d = sin(qα/2)/(q sin(α/2))
短距系数 k_p k_p 反映绕组短距的影响 k_p = sin(y/τ × 90°)

8.4 三相交流绕组

三相单层绕组：

  • 每个槽只放一个线圈边
  • 线圈数 = 槽数/2
  • 分为等元件式、交叉式（同心式）、链式
  • 一般用于小型异步电动机定子绕组

三相双层绕组：

  • 每个槽放两个线圈边（上层和下层）
  • 线圈数 = 槽数
  • 分为叠绕组和波绕组
  • 叠绕组：汽轮发电机及大中型异步电动机定子
  • 波绕组：水轮发电机定子及绕线式异步电动机转子
  • 实际中广泛采用双层短距叠绕组

绕组构成原则：

  1. 产生的合成电动势和磁动势波形接近正弦波（谐波分量尽可能小）
  2. 在一定导体数下，得到较大的基波电动势和磁动势
  3. 三相绕组对称（大小相等、相位差120°、阻抗相等）
  4. 铜耗小，用铜量少
  5. 绝缘可靠，机械强度高，散热条件好

8.5 分数槽绕组（带星号，可能不考）

8.6 正弦分布磁场中交流绕组的感应电动势

整距集中绕组电动势：

E=4.44fNΦmE = 4.44 f N \Phi_m E=4.44fNΦm​

考虑分布和短距后的电动势：

E=4.44fNkwΦmE = 4.44 f N k_w \Phi_m E=4.44fNkw​Φm​

其中 k_w = k_d × k_p 为绕组系数。

关键公式：

  • 同步转速：n₁ = 60f/p
  • 频率：f = pn₁/60

8.7 非正弦磁场中交流绕组电动势及减小谐波电动势的方法

谐波电动势产生原因： 磁场非正弦分布

减小谐波电动势的方法：

  1. 采用短距绕组（选择适当节距）
  2. 采用分布绕组（增加每极每相槽数q）
  3. 采用斜槽

8.8 正弦电流时交流绕组的磁动势

单相绕组磁动势——脉振磁动势

单相交流绕组通入正弦电流产生脉振磁动势：

f(x,t)=Fmcos⁡x⋅cos⁡ωtf(x,t) = F_m \cos x \cdot \cos \omega t f(x,t)=Fm​cosx⋅cosωt

脉振磁动势可分解为两个幅值相等、转向相反的旋转磁动势。

三相绕组的合成磁动势——旋转磁动势

三相对称绕组通入三相对称电流产生圆形旋转磁动势：

特点：

  1. 幅值恒定：F1=1.35NkwpIF_1 = 1.35 \frac{N k_w}{p} IF1​=1.35pNkw​​I（为单相脉振磁动势幅值的3/2倍）
  2. 转速：同步转速 n1=60f/pn_1 = 60f/pn1​=60f/p（r/min）
  3. 转向：从电流超前相转向电流滞后相（改变相序可反转）
  4. 幅值位置：某相电流最大时，合成磁动势幅值与该相轴线重合

旋转磁场的两种类型（教材补充）：

  • 机械旋转磁场：磁场本身恒定，由原动机拖动磁极旋转（如同步电机转子）
  • 电气旋转磁场：三相对称交流绕组通入三相对称交流电流产生（如同步/异步电机定子）

两相绕组的磁动势

  • 对称两相绕组（空间差90°电角度，通入时间差90°电流）→ 圆形旋转磁动势
  • 不对称两相绕组 → 椭圆形旋转磁动势

总结：

  • 当正转分量≠0，反转分量=0 → 圆形旋转磁动势
  • 当正转分量≠反转分量 → 椭圆形旋转磁动势
  • 当正转分量=反转分量 → 脉振磁动势

教材本章小结（重要）：
本章讨论了交流同步电机和异步电机的共同部分——交流绕组。首先介绍了交流绕组的构成原则、类型及相关的基本概念，然后讨论了三相单层绕组和三相双层绕组的连接，之后分别讨论了正弦磁场下交流绕组的感应电动势和正弦电流下交流绕组的磁动势。
正弦磁场下一相交流绕组感应电动势计算公式与变压器绕组感应电动势的计算公式类似，只是由于交流绕组为分布式、短距绕组，所以感应电动势计算公式中的绕组匝数为等效匝数，即绕组实际匝数乘以绕组系数。单相交流绕组流过正弦电流产生脉振磁场，空间对称分布的三相交流绕组通以三相对称电流产生旋转磁场，旋转磁场的速度称为同步转速，是电机的重要参数。
