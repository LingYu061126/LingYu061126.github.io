变压器标幺值、参数测定与运行特性

一、标幺值 (Per Unit System)

1. 标幺值的定义

标幺值 = 实际值 / 基准值（同单位）

  • 相对值，无单位
  • 用右下角加"*"表示

2. 基准值的选取

物理量 基准值选取
电压/电流 额定相电压/电流（相量）；额定线电压/电流（线量）
阻抗（R, X, Z） 额定相电压与额定相电流的比值
功率（P, Q, S） 额定视在功率 SNS_NSN​
一、二次侧 各自选取本侧的额定值

3. 标幺值的特点

  1. 额定值的标幺值为1：UN∗=IN∗=SN∗=1U_N^* = I_N^* = S_N^* = 1UN∗​=IN∗​=SN∗​=1
  2. 折算前后标幺值相等：U2∗=U2′∗U_{2}^* = U_{2}'^*U2∗​=U2′∗​，计算时不必折算
  3. 短路阻抗标幺值 = 阻抗电压标幺值：Zk∗=Uk∗Z_k^* = U_k^*Zk∗​=Uk∗​
  4. 单相和三相标幺值相等

二、变压器等效电路参数测定

1. 空载试验 (Open Circuit Test)

项目 内容
接线方式 低压侧加电压，高压侧开路
测量参数 变比 K、空载电流 I0I_0I0​、空载损耗 P0P_0P0​、励磁参数
关键注意 应取额定电压点计算励磁参数（因励磁阻抗与铁心饱和程度有关）
损耗近似 空载损耗 ≈ 铁耗 P0≈PFeP_0 \approx P_{Fe}P0​≈PFe​

计算公式：

K=U20U1N,rm=P0I02,Zm=U1NI0,xm=Zm2−rm2K = \frac{U_{20}}{U_{1N}}, \quad r_m = \frac{P_0}{I_0^2}, \quad Z_m = \frac{U_{1N}}{I_0}, \quad x_m = \sqrt{Z_m^2 - r_m^2} K=U1N​U20​​,rm​=I02​P0​​,Zm​=I0​U1N​​,xm​=Zm2​−rm2​​

> 注意：励磁参数为低压侧值，要得到高压侧参数需乘以 K2K^2K2

2. 短路试验 (Short Circuit Test)

项目 内容
接线方式 高压侧加电压，低压侧短路
试验条件 在较低电压下进行，短路电流达到额定值为止（外加电压约为额定电压的5-10%）
测量参数 短路参数及铜耗
关键注意 必须测量环境温度，换算到标准工作温度75°C
损耗近似 短路损耗 ≈ 铜耗 Pk≈PCuP_k \approx P_{Cu}Pk​≈PCu​（铁耗忽略）

计算公式：

Zk=UkIk,rk=PkIk2,xk=Zk2−rk2Z_k = \frac{U_k}{I_k}, \quad r_k = \frac{P_k}{I_k^2}, \quad x_k = \sqrt{Z_k^2 - r_k^2} Zk​=Ik​Uk​​,rk​=Ik2​Pk​​,xk​=Zk2​−rk2​​

rk75°C=rkθ⋅235+75235+θ,Zk75°C=rk75°C2+xk2r_{k75°C} = r_{k\theta} \cdot \frac{235+75}{235+\theta}, \quad Z_{k75°C} = \sqrt{r_{k75°C}^2 + x_k^2} rk75°C​=rkθ​⋅235+θ235+75​,Zk75°C​=rk75°C2​+xk2​​

r1=r2′=rk2,x1=x2′=xk2r_1 = r_2' = \frac{r_k}{2}, \quad x_1 = x_2' = \frac{x_k}{2} r1​=r2′​=2rk​​,x1​=x2′​=2xk​​

3. 阻抗电压 (Impedance Voltage)

定义： 短路试验时，使短路电流达到额定值所加电压，用与额定电压的百分比表示：

uk=UkU1N×100%u_k = \frac{U_k}{U_{1N}} \times 100\% uk​=U1N​Uk​​×100%

分量 物理意义 标幺值表示
ukau_{ka}uka​（有功分量） 平衡短路电阻压降 Uka∗=rk∗U_{ka}^* = r_k^*Uka∗​=rk∗​
ukru_{kr}ukr​（无功分量） 平衡短路电抗压降 Ukr∗=xk∗U_{kr}^* = x_k^*Ukr∗​=xk∗​

典型取值：

  • 中小型变压器：uk=4%∼10.5%u_k = 4\% \sim 10.5\%uk​=4%∼10.5%
  • 大型变压器：uk=12.5%∼17.5%u_k = 12.5\% \sim 17.5\%uk​=12.5%∼17.5%

三、变压器运行特性

1. 外特性 (电压调整特性)

定义： 一次侧为额定电压、负载功率因数不变时，U2=f(I2)U_2 = f(I_2)U2​=f(I2​) 的曲线

电压变化率：

ΔU=U2N−U2U2N=1−U2∗\Delta U = \frac{U_{2N}-U_2}{U_{2N}} = 1-U_2^* ΔU=U2N​U2N​−U2​​=1−U2∗​

实用计算公式（简化）：

ΔU≈β(rk∗cos⁡φ2+xk∗sin⁡φ2)\Delta U \approx \beta(r_k^* \cos\varphi_2 + x_k^* \sin\varphi_2) ΔU≈β(rk∗​cosφ2​+xk∗​sinφ2​)

其中 β=I2/I2N\beta = I_2/I_{2N}β=I2​/I2N​ 为负载系数

负载特性影响：

负载类型 功率因数 电压变化 外特性曲线
纯电阻负载 cos⁡φ=1\cos\varphi=1cosφ=1 电压略降 轻微下倾
感性负载 cos⁡φ\cos\varphicosφ滞后 电压下降明显 下降曲线
容性负载 cos⁡φ\cos\varphicosφ超前 电压可能上升 上升曲线（可能高于额定电压）

2. 效率特性

损耗组成

损耗类型 组成 特点 与负载关系
铜耗 PCuP_{Cu}PCu​ 基本铜耗（直流电阻）+ 附加铜耗（集肤效应、涡流等） 可变损耗 与电流平方成正比：PCu=β2PkNP_{Cu} = \beta^2 P_{kN}PCu​=β2PkN​
铁耗 PFeP_{Fe}PFe​ 基本铁耗（磁滞、涡流）+ 附加铁耗 不变损耗 不随负载变化：PFe≈P0P_{Fe} \approx P_0PFe​≈P0​

η=βSNcos⁡φ2βSNcos⁡φ2+P0+β2PkN×100%\eta = \frac{\beta S_N \cos\varphi_2}{\beta S_N \cos\varphi_2 + P_0 + \beta^2 P_{kN}} \times 100\% η=βSN​cosφ2​+P0​+β2PkN​βSN​cosφ2​​×100%

最大效率条件

当铁耗 = 铜耗时（不变损耗 = 可变损耗），效率最高：

βm=P0PkN\beta_m = \sqrt{\frac{P_0}{P_{kN}}} βm​=PkN​P0​​​

ηmax=βmSNcos⁡φ2βmSNcos⁡φ2+2P0×100%=[1−2P0βmSNcos⁡φ2+2P0]×100%\eta_{max} = \frac{\beta_m S_N \cos\varphi_2}{\beta_m S_N \cos\varphi_2 + 2P_0} \times 100\% = \left[1-\frac{2P_0}{\beta_m S_N \cos\varphi_2 + 2P_0}\right] \times 100\% ηmax​=βm​SN​cosφ2​+2P0​βm​SN​cosφ2​​×100%=[1−βm​SN​cosφ2​+2P0​2P0​​]×100%

设计特点：

  • 变压器设计时取 P0/PkN≈1/4∼1/3P_0/P_{kN} \approx 1/4 \sim 1/3P0​/PkN​≈1/4∼1/3
  • 最高效率发生在负载系数 β=0.5∼0.6\beta = 0.5 \sim 0.6β=0.5∼0.6 范围内（而非额定负载）
  • 中小型变压器效率：95%~98%；大型变压器可达99%
