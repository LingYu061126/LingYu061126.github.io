微积分知识整理 定积分证明(1) (未校对)

发表于2026-02-25|更新于2026-02-25|学习笔记
|浏览量:

一、直接构造变限函数

例 1（四川大学2023-2024级考期末）设 ( f(x) ) 在 ([a, a]) 上非负，且 ( f(0) = 0 )，( f’(x) > 0 )。证明：
[
\int_0^a x f(x) , dx > \frac{2}{3} a \int_0^a f(x) , dx
]
（注：原题区间可能为 ([0,a])，此处保留笔记原文 ([a, a])，但根据后续推导应为 ([0,a])。）

证明：构造函数
[
F(x) = \int_0^x t f(t) , dt - \frac{3}{2} x \int_0^x f(t) , dt
]
求导得
[
F’(x) = x f(x) - \frac{3}{2} \int_0^x f(t) , dt - \frac{3}{2} x f(x) = \frac{3}{2} x f(x) - \frac{3}{2} \int_0^x f(t) , dt
]
再求导
[
F’‘(x) = \frac{3}{2} f(x) + \frac{3}{2} x f’(x) - \frac{3}{2} f(x) = \frac{3}{2} x f’(x) \ge 0
]
（此处原笔记有笔误，根据导数计算应为 (F’‘(x) = \frac{3}{2} x f’(x))，因 (f’(x)>0) 且 (x\ge0) 故非负。）
当且仅当 (x=0) 时 (F’'(x)=0)。而 (a>0)，(f(x)\ge0)，(\int_0^a f(x)dx\ge0)，从而有 (F(a)>F(0)=0)，即
[
\int_0^a t f(t) , dt > \frac{3}{2} a \int_0^a f(t) , dt
]
将 (t) 换为 (x) 即得欲证不等式。

例 2（清华2018）设函数 (f(x), g(x)) 在 ([a, b]) 上连续，对任意 (x \in [a, b]) 有
[
\int_a^x g(t) , dt < \int_a^x f(t) , dt
]
且 (\int_a^b g(t) , dt = \int_a^b f(t) , dt)。求证：
[
\int_a^b x f(x) , dx < \int_a^b x g(x) , dx
]

证明：令 (E(x) = f(x) - g(x))，(G(x) = \int_a^x f(t) , dt)，由条件 (\int_a^x g(t) dt < \int_a^x f(t) dt) 知 (G(x) \ge 0)。
构造并利用分部积分：
[
\int_a^b x g(x) , dx - \int_a^b x f(x) , dx = -\int_a^b x \bigl(f(x)-g(x)\bigr) dx = -\int_a^b x , dG(x)
]
（其中 (G(x) = \int_a^x (f(t)-g(t)) dt)，注意符号）
[
-\int_a^b x , dG(x) = -\left[ x G(x) \Big|_a^b - \int_a^b G(x) dx \right] = -b G(b) + \int_a^b G(x) dx
]
由于 (G(b) = \int_a^b (f-g) dx = 0)，故上式 (= \int_a^b G(x) dx \ge 0)。因此
[
\int_a^b x g(x) , dx \ge \int_a^b x f(x) , dx
]
等号成立仅当 (G(x) \equiv 0)，即 (f=g)，与已知严格不等式矛盾，故严格小于号成立。Q.E.D.

总结：这两题的特点为出现了积分上限一致、被积函数不同的待证式，一般均可考虑将其中的上限换为自变量构造变限函数。

二、先对被积函数泰勒展开再两端积分（或直接积分）

例 1（四川大学2022-2023秋季期末）设 (f(x)) 在 ([0,1]) 上二阶连续可导，(f’(x) \ge 0)，(f’'(x) < 0)。令
[
I_n = \int_0^1 f(x^n) , dx \quad (n \in \mathbb{N})
]
证明：
[
I_n \le f\left(\frac{1}{n}\right)
]

证明：将 (f(x)) 在 (x = \frac{1}{n}) 处展开得
[
f(x) = f\left(\frac{1}{n}\right) + f’\left(\frac{1}{n}\right)\left(x - \frac{1}{n}\right) + \frac{f’‘(\xi)}{2!}\left(x - \frac{1}{n}\right)^2
]
于是
[
f(x^n) = f\left(\frac{1}{n}\right) + f’\left(\frac{1}{n}\right)\left(x^n - \frac{1}{n}\right) + \frac{f’‘(\xi)}{2!}\left(x^n - \frac{1}{n}\right)^2
]
积分得
[
I_n = f\left(\frac{1}{n}\right) + f’\left(\frac{1}{n}\right) \int_0^1 \left(x^n - \frac{1}{n}\right) dx + \int_0^1 \frac{f’‘(\xi)}{2!}\left(x^n - \frac{1}{n}\right)^2 dx
]
由于 (\int_0^1 \left(x^n - \frac{1}{n}\right) dx = \frac{1}{n+1} - \frac{1}{n} < 0)，而 (f’\left(\frac{1}{n}\right) \ge 0)，故第二项 (\le 0)；又 (f’'(\xi) < 0)，第三项 (\le 0)。因此
[
I_n \le f\left(\frac{1}{n}\right)
]
Q.E.D.

注：当 (n=2) 时即得下面特例。

例 2 设 (f(x) \in D^2[0,1])，且 (f’(x) < 0)。证明：
[
\int_0^1 f(x) , dx \le f\left(\frac{1}{2}\right)
]
（此即例1中 (n=2) 的情形。）

例 3 设 (f(x) \in D^2[0,1])，证明当 (\exists a,b \in (a,b)) 时（此处原文不清，可能意为 (a<b)），
[
\int_a^b f(x) , dx = (b-a) f\left(\frac{a+b}{2}\right) + \frac{(b-a)^3}{24} f’‘(\xi)
]
证明：将 (f(x)) 在 (x = \frac{a+b}{2}) 处展开：
[
f(x) = f\left(\frac{a+b}{2}\right) + f’\left(\frac{a+b}{2}\right)\left(x - \frac{a+b}{2}\right) + \frac{f’‘(\xi)}{2}\left(x - \frac{a+b}{2}\right)^2
]
两端积分，注意到 (\int_a^b \left(x - \frac{a+b}{2}\right) dx = 0)，且
[
\int_a^b \left(x - \frac{a+b}{2}\right)^2 dx = \frac{(b-a)^3}{12}
]
因此
[
\int_a^b f(x) , dx = (b-a) f\left(\frac{a+b}{2}\right) + \frac{f’‘(\xi)}{2} \cdot \frac{(b-a)^3}{12} = (b-a) f\left(\frac{a+b}{2}\right) + \frac{(b-a)^3}{24} f’'(\xi)
]
Q.E.D.

提示：注意，有的时候并不一定要先将积分中被积函数展开，下举两例说明。

例 4 设 (f(x) \in C[0,+\infty))，(g(x) = \int_0^x f(t) , dt)，广义积分 (\int_0^\infty f(x) , dx) 收敛。证明：
[
\int_0^\infty \frac{g(x)}{x^2} , dx < 4 \int_0^\infty f(x) , dx
]
（原笔记中写为 (\int_0^\infty g(x) dx < 4 \int_0^\infty f(x) dx)，可能笔误，应为 (\int_0^\infty \frac{g(x)}{x^2} dx)，因后续推导涉及 (\frac{g(x)}{x})。）

证明思路：利用分部积分和Cauchy-Schwarz不等式。先证 (\frac{g(x)}{x}) 有界，并利用
[
\int_0^\infty \frac{g(x)}{x^2} dx = \int_0^\infty \frac{1}{x^2} \int_0^x f(t) dt , dx
]
交换积分次序或使用Cauchy不等式放缩。原笔记过程较乱，摘录如下：

“证明 (\frac{g(x)}{x}) 有上界
[
\int_0^x \frac{g(t)}{t} dt, \quad \lim_{x \to \infty} \frac{g(x)}{x} = \lim_{x \to \infty} \int_0^x \frac{g(t)}{t} dt = \int_0^\infty f(x) dx
]
从右端 (x \to \infty), (g(x) \geq 0), 从而 (\frac{g(x)}{x} \geq 0)。
(\frac{g(x)}{x} \leq 0) 故缩
从而 ((\int_0^x \frac{g(t)}{t})^2 \leq (2 \int_0^x \frac{f(t)}{t})^2 \leq 4 \int_0^x f^2(x))。
再次使用 Cauchy 不等式
[
\int_0^x \frac{g(t)}{t} \leq 4 \int_0^x f^2(x)
]
比较审敛法知原式积分收敛。 (Q.E.D.)”

此处保留原笔记，但显然有混乱，我们按原样呈现。

例 5 设 (f(x) \in C[0,1])，且满足
[
|f(x)| \le 1 + \int_0^x |f(t)| , dt, \quad \forall x \in [0,1]
]
证明：(|f(x)| \le e^x)。

证明：令 (g(x) = \int_0^x |f(t)| , dt)，则 (g’(x) = |f(x)|)，且由条件得
[
g’(x) \le 1 + g(x)
]
解此一阶线性微分不等式：((g(x)e^{-x})’ \le e^{-x})，积分得
[
g(x)e^{-x} \le \int_0^x e^{-t} dt + C
]
取 (C=0)（因 (g(0)=0)），则
[
g(x) \le e^x \int_0^x e^{-t} dt = e^x (1 - e^{-x}) = e^x - 1
]
但我们需要的是 (|f(x)|) 的界，由 (g’(x) \le 1 + g(x) \le 1 + (e^x - 1) = e^x)，故 (|f(x)| = g’(x) \le e^x)。Q.E.D.

总结：当待证式中仅出现一个积分且有高阶导数时，即可考虑先展开被积函数再两端积分（一般亦可直接展开变限函数，但相对繁琐）。当不符合高阶导数时，亦可考虑直接积分或换元，将积分不等式转为非积分部分（参见后续步骤）。
